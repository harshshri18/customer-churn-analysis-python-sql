📊 Customer Churn Analysis using Python & SQL
Project Overview

This project analyzes customer churn for an OTT subscription platform using Python and SQL. The objective is to identify customers likely to churn, understand churn behavior, calculate business KPIs, and provide actionable insights to improve customer retention.

Business Problem

Customer churn directly impacts revenue and profitability. This project identifies:

Customers likely to churn

Reasons behind churn

Revenue loss

Customer lifetime value (CLTV)

Retention opportunities
